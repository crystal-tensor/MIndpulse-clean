## 🎨 截图技巧

### 浏览器设置
- 使用Chrome或Safari浏览器
- 设置窗口大小为1920x1080或更高分辨率
- 确保界面完整显示，避免滚动条
- 关闭开发者工具和其他干扰元素

### 截图工具推荐
- **macOS**: 使用 `Cmd + Shift + 4` 进行区域截图
- **Windows**: 使用 `Win + Shift + S` 或 Snipping Tool
- **浏览器扩展**: Awesome Screenshot, Lightshot

### 图片处理
- 保存为PNG格式以保证清晰度
- 文件大小控制在2MB以内
- 可以适当添加标注和说明文字
- 统一图片风格和尺寸比例
